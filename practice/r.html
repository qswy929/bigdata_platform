RStudio的使用教程

RStudio简介

本平台的RStudio依托于星环TOS中的Discover机器学习组件，提供多种传统及分布式的算法包。下面将以Iris鸢尾花卉数据为例，简单介绍SQL编辑器的使用方法。

使用教程

1. 选择数据集
选择导航栏中的“数据集”，可以看到平台中提供的所有数据集，点选需要的数据集，如Iris数据。
2. 进入RStudio
默认将会跳转到SQL编辑器的界面，此时可以点击右上角的下拉菜单，选择"R"，将会切换到RStudio的界面，如下图。
3. 数据分析
首先使用其中的"bigdataR"包将Iris数据从Inceptor中加载：
sc <- bigdata.init() #初始化连接 sqlCon <- txConnect(host = 'inceptor23i1ao-1232460318-97k0n', dbName = 'iris') #使用该host和dbName table <- txSqlQuery(sqlConnection = sqlCon, query = 'select id, sepallengthcm, sepalwidthcm, petallengthcm, petallengthcm, species from iris') #SQL语句" iris_demo <- txCollect(list(table$id,table$sepallengthcm,sepalwidthcm,table$petallengthcm,table$petallengthcm,table$species)) #选择所有列 txRemove(table) txSqlDisconnect(sqlConnection = sqlCon) #关闭连接
